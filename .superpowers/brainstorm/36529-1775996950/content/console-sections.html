What's on the Cross-Platform tab?

All data exists in the DB. Pick which sections are in scope for MVP.

1

KPI row — 4 cards

Total cross-tool spend · Top tool by cost · Active developers · MTD vs budget

2

Tool breakdown chart

Bar or donut: Copilot vs Claude Code vs Cursor vs Gemini CLI cost share. Period selector (7/30/90d).

3

Per-developer attribution table

email · total cost · tools used · commits · cost/commit. Sortable. Click row → drill-down modal.

4

Daily cost trend (stacked by tool)

Line/area chart showing per-tool spend over time. Spots when Cursor started spiking, etc.

5

Live event feed

Real-time stream of API calls across all tools. Already exists on Overview — include here too?

6

Connected tools status

Which billing connectors are active, last sync time, errors. Quick "Add tool" CTA.
