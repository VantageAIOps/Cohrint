AI Spend Console — Where does it live?

The console needs a home in the dashboard. Pick the layout that fits best.

A

New top-level nav section

Sidebar gets a dedicated "AI Spend Console" entry at the same level as Dashboard, Team, Settings. Full-page view: 4 KPI cards + tool breakdown chart + per-developer table + live feed. Named product, standalone.

Best for: enterprise CTOs who want a dedicated cost command center
B

New tab inside Dashboard

Dashboard section gets a "Cross-Platform" tab alongside existing Overview / Spend Analysis / Model Pricing tabs. Same data, lighter navigation weight. Keeps dashboard as the single hub.

Best for: power users already living in the dashboard
C

Expand Overview + add dedicated section

Keep Overview as the landing page, but add a prominent "AI Spend Console" section below current KPIs. No new nav item — just scroll down. Lower discovery risk, zero nav changes.

Best for: fast ship, minimal UI surgery
