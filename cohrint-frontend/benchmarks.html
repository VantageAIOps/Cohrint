How This Data Works
Privacy-first benchmark methodology
🔐 K-Anonymity
Cohorts with fewer than 5 contributing organizations return no data. Your org is never identifiable in the aggregate.
📈 Quarterly Snapshots
Data is aggregated quarterly. Percentiles are computed from all opted-in orgs in each size band and recalculated each Sunday.
✅ Opt-in Only
Only organizations that explicitly enable benchmark sharing contribute data. Participation is always voluntary and can be revoked.
