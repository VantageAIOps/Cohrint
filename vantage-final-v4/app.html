How Token Optimization Works
1. SDK Integration

Enable optimizer in your SDK init: vantage.init(api_key="...", enable_optimizer=True). All subsequent LLM calls are automatically optimized.

2. Prompt Compression

Redundant phrases, filler words, and excessive whitespace are removed while preserving semantic meaning. Advanced mode uses LLMLingua for ML-based compression.

3. Context Management

Long conversation contexts are automatically summarized. Older messages are compressed while recent messages remain verbatim.

4. Cost Tracking

Every optimized call tracks original vs compressed tokens, so you see exact savings in the dashboard above.
